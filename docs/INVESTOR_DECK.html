Seccion 02
El Producto
Que es TallyFinance y como funciona.

Gus: tu companero financiero

Gus no es un chatbot generico. Es un personaje con identidad chilena, tono personalizable (amigable, estricto, roaster, motivacional), que entiende "lucas", chilenismos, y te habla como un amigo que sabe de plata. Vive en la app, la web, Telegram y WhatsApp.

Los 3 pilares

Pilar 01 — Ref: Cleo ($350M ARR)

Tracking Automatico

Conexion bancaria directa via Fintoc (9 bancos chilenos). Gus ve todas las transacciones, las categoriza con ML (identificando comercios, suscripciones y patrones), y envia alertas cuando te acercas a limites de presupuesto. Cero registro manual.

Integrado con Fintoc
Pilar 02 — Ref: Qapital (rules engine)

Ahorro Gamificado

Motor de reglas automaticas: redondeo de compras, "guilty pleasure" (gastas en X → ahorras Y), challenge de 52 semanas, ahorro por porcentaje del sueldo. Metas visuales con barras de progreso. Todo configurable via chat con Gus.

Sobre data real del banco
Pilar 03 — Diferenciador propio

Coaching con Personalidad

Roast mode (Gus te destruye por tus malos habitos), toast mode (te felicita por ahorrar), resumenes semanales, deteccion de suscripciones olvidadas. Cuatro tonos de personalidad. Contenido compartible y viral.

Seccion 04
Open Banking en Chile
El catalizador regulatorio que define el timing.

El Sistema de Finanzas Abiertas (SFA)

La Ley 21.521 (Ley Fintech) y la NCG N.514 de la CMF obligan a todos los bancos, cooperativas, aseguradoras y AFP a abrir sus datos via APIs estandarizadas. Esto crea un mercado completamente nuevo de servicios financieros basados en datos.

Por que importa para TallyFinance

Hoy: Conexion via Fintoc (pre-Open Banking)

Ya operamos con 9 bancos chilenos via Fintoc. Vemos transacciones de cuentas corrientes y vista (debito, PAC, TEF). Esto cubre la mayoria del gasto del segmento universitario (CuentaRUT + debito).

2027+: APIs oficiales del SFA

Cuando el SFA entre en vigencia, accederemos a TODAS las transacciones (incluyendo tarjetas de credito), de TODOS los bancos, via APIs estandarizadas con OAuth 2.0. La cobertura pasa de 70% a 100%.

Ventaja de timing

Estamos construyendo la infraestructura ANTES de que Open Banking sea obligatorio. Cuando las APIs oficiales se abran, ya tendremos producto validado, usuarios activos, ML entrenado, y marca posicionada. Los competidores que empiecen en 2027 llegaran 18+ meses tarde.

Moat (fosos competitivos)

1. Gus como activo de marca

Personalidad irrecplicable. Cleo invierte en 15 comediantes para mantener su tono. Gus es el equivalente chileno: entiende lucas, chilenismos, y genera contenido compartible (roasts). La personalidad es el producto, no una feature.

2. Data advantage compuesto

Cada transaccion que procesamos entrena el ML de categorizacion. Mas usuarios → mejor parser → mejor experiencia → mas usuarios. Este ciclo es dificil de replicar para un competidor que empiece despues.

3. First-mover en Open Banking

Cuando el SFA sea obligatorio (2027), seremos la unica app con producto validado, ML entrenado, y usuarios activos. Los bancos tendran features basicas de PFM en sus apps, pero sin personalidad ni ahorro gamificado.

4. Costo de cambio

Historial de transacciones categorizadas, metas en progreso, reglas de ahorro configuradas, personalidad calibrada. Todo genera lock-in organico sin contratos.

Orquestacion IA (Two-Phase)

El sistema usa un patron de doble fase para maximizar precision y personalidad:

Fase A — Entender

Analiza el mensaje del usuario, identifica intencion, y decide que herramienta ejecutar (registrar gasto, consultar balance, etc.). Modelo con temp=0.3 para precision.

Fase B — Responder

Toma el resultado de la herramienta + contexto del usuario (mood, estilo, historial) y genera la respuesta con la personalidad de Gus. Temp=0.7 para creatividad.

ML para categorizacion de transacciones

Fintoc devuelve transacciones con un campo description de texto libre del banco (ej: "COMPRA NAC 05/04 LIDER"). Construimos un parser ML que convierte esto en:

Input (description del banco)Output: MerchantOutput: Categoria
"COMPRA NAC 05/04 LIDER"LiderSupermercado
"CARGO AUTOMATICO SPOTIFY"SpotifySuscripcion
"COMPRA INT UBER EATS"Uber EatsDelivery
"PAC ENTEL"EntelTelefonia

Referencia: Copilot Money ($4.1M ARR) logra 93% de precision en categorizacion automatica con ML per-user. Es un problema resuelto en la industria.

Datos de la API Fintoc (validado en sandbox)

CampoTipoQue nos da
