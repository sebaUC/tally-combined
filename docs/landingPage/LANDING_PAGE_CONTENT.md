# 🎯 TallyFinance — Landing Page Content Blueprint

> **Framework:** PAS + BAB Hybrid (Problem → Agitate → Solution + Before → After → Bridge)
> **Total copy target:** 500–700 words across all sections
> **Language:** Spanish (Chilean)
> **Conversion goal:** User clicks → opens WhatsApp/Telegram → sends first message
